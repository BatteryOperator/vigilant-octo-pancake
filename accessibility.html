Accessibility Statement

Last Updated: January 8, 2026

Our Commitment to Accessibility

Blub Blub Studios LLC is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

Conformance Status

We aim to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA. These guidelines explain how to make web content more accessible for people with disabilities and user-friendly for everyone.

Measures to Support Accessibility

Blub Blub Studios takes the following measures to ensure accessibility:

  • Include accessibility as part of our design and development process
  • Provide clear and simple language throughout our content
  • Ensure proper color contrast ratios for readability
  • Use semantic HTML for better screen reader compatibility
  • Provide text alternatives for non-text content
  • Enable keyboard navigation throughout the site
  • Use ARIA landmarks and labels where appropriate
  • Test with assistive technologies

Accessibility Features

This website includes the following accessibility features:

Navigation

  • Skip to Content Link - Keyboard users can press Tab to reveal a "Skip to main content" link
  • Keyboard Navigation - All interactive elements are accessible via keyboard
  • Clear Focus Indicators - Visible focus states for all interactive elements
  • Semantic Structure - Proper headings and landmarks for screen reader users

Design

  • Color Contrast - Text meets WCAG AA contrast requirements (4.5:1 minimum)
  • Readable Fonts - Clear, professional typography optimized for readability
  • Responsive Design - Works on all screen sizes and zoom levels up to 200%
  • Minimal Distraction - Clean layouts without unnecessary animations or distractions

Content

  • Alternative Text - All images include descriptive alt text
  • Clear Language - Content written in plain, straightforward language
  • Logical Reading Order - Content follows a natural reading sequence
  • Descriptive Links - Link text clearly describes destination

Known Limitations

Despite our best efforts to ensure accessibility, there may be some limitations. We are committed to addressing these issues:

  • We continuously monitor and improve our accessibility practices
  • User feedback helps us identify and fix accessibility barriers

Feedback and Contact

We welcome your feedback on the accessibility of the Blub Blub Studios website. If you encounter any accessibility barriers, please let us know:

Accessibility Concerns

Email: support@blubblubstudios.com

Please include:

  • The page where you experienced the issue
  • A description of the accessibility barrier
  • What assistive technology you were using (if applicable)

We aim to respond to accessibility feedback within 2 business days.

Technical Specifications

This website's accessibility relies on the following technologies:

  • HTML5
  • CSS3
  • JavaScript (for mobile menu functionality - gracefully degrades without JavaScript)

Assessment Approach

Blub Blub Studios assessed the accessibility of this website through:

  • Self-evaluation using WCAG 2.1 guidelines
  • Automated testing tools
  • Manual testing with keyboard navigation
  • Color contrast verification

Third-Party Content

This website does not embed third-party content. All content is directly controlled by Blub Blub Studios to ensure accessibility standards are maintained.

Our Apps

Our mobile applications, including Story Slider, are also designed with accessibility in mind. We follow Apple's Human Interface Guidelines for iOS accessibility and ensure our apps work with VoiceOver and other assistive technologies.
